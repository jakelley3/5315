What is Docker

Docker is a tool that unites all parts of the application and ships it out as a single application for other Linux machines. It does this through the usage of online containers. Furthermore, Docker is an open source application, meaning users can customize applications to fit additional business needs.

Source: https://opensource.com/resources/what-docker

What is Nginx

Nginx is an open source service that was initially created as a high perfromance web server. While it is still used in this capacity, it can also be used as a proxy server for email and a reverse proxy and load balancer for various servers. The goal of Nginx is to offer the fastest web server capabilities in the industry.

Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A virtual machine is a computer file that allows users to access and work on the host operating system from remote locations. Since a virtual machine is a seperate entity outside of the host operating system, it is ideal for testing other operating systems.

Source: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

Digital Ocean offers cloud computing solutions to users through a IaaS offering (Infrastructure as a Service). Users can launch a virtual machine instance (droplet) to deploy the company's cloud computing infrastructure for personal use. DigitalOceean was founded in 2011 by Ben Uretsky, Moisey Uretsky, Mitch Wainer, Jeff Carr and Alec Hartman.

Source: https://searchcloudcomputing.techtarget.com/definition/DigitalOcean
